CSS Color

Goal: Explain how CSS uses color and show examples for color, background-color, and border-color. This page also demonstrates several ways to write color values.

What do these properties do?

  • color — sets the color of text (the “foreground” of a font).
  • background-color — fills the background of an element’s box.
  • border-color — colors the border lines around an element.
This box uses color: rebeccapurple; for the text.
This box uses background-color: #fef08a; (hex).
This box uses border-color: rgb(34, 197, 94); (RGB).

How CSS accepts color values (with examples)

CSS accepts many formats. The most common are named colors, hexadecimal, RGB/RGBA, and HSL/HSLA. Below are small swatches showing each format. You can copy and paste these values.

Named

rebeccapurple, tomato, goldenrod

Hexadecimal

#1f2937, #f59e0b, #10b981

RGB / RGBA

rgb(59, 130, 246)
rgba(59, 130, 246, 0.35)

HSL / HSLA

hsl(210 20% 20%)
hsla(14, 90%, 50%, 0.8)

This box mixes formats: background-color: rgba(59,130,246,.15) and text color: hsl(14 90% 40%).
