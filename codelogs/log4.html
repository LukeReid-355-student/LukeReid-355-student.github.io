CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media query lets developers apply styles only when certain conditions about the device or browser window are true. For example, you can write styles that only apply if the screen is less than 768px wide or if the device is in portrait orientation. This is useful for creating responsive designs that adjust automatically to desktops, tablets, and smartphones, user experience.

What Media Features are supported by Media Queries?

Media queries support many features, but here are four important ones:

1. Max Width

Targets screens that are at or below a certain width, making it useful for mobile-friendly layouts.

2. Orientation

Detects whether the device is in portrait (taller than wide) or landscape (wider than tall).

3. Prefers Color Scheme

Checks if the user prefers light mode or dark mode and allows your website to adapt automatically.

4. Resolution

Targets high-density displays (like Retina screens) so you can provide sharper images or icons.
